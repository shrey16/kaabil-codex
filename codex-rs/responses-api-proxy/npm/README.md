# @kaabil/codex-responses-api-proxy

<p align="center">This fork does not publish npm packages. Build from source in <code>codex-rs/responses-api-proxy</code>.</p>

This directory contains the npm wrapper for the [Codex Responses API proxy binary](https://github.com/shrey16/kaabil-codex/tree/main/codex-rs/responses-api-proxy). The Kaabil fork does not publish this package.

To see available options after building, run:

```
codex-responses-api-proxy --help
```

Refer to [`codex-rs/responses-api-proxy/README.md`](https://github.com/shrey16/kaabil-codex/blob/main/codex-rs/responses-api-proxy/README.md) for detailed documentation.
